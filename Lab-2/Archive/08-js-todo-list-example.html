Exercise 8

Javascript to do list อย่างง่าย

To do list

  • ทำการบ้าน web programming
  • ซื้อข้าวหมา
  • จ่ายค่าโทรศัพท์
  • จ่ายค่าหอ
  • คิดไม่ออก

โจทย์

  • สามารถลบรายการได้
  • สามารถเพิ่มรายการใหม่ได้

ลับสมอง

  • สบายๆ สามารถเพิ่มรายการใหม่ได้ โดยพิมพ์ข้อความใน textbox แล้วกด enter
  • ยากนิดหน่อย สมมติว่าดึงข้อมูลจาก API ได้รับขัอมูลเป็น javascript array
    let todos = ['task1', 'task2', 'task3', 'task4']

    โดย To do list จะต้องแสดงผลตามข้อมูลที่ได้จาก API ดังนี้
