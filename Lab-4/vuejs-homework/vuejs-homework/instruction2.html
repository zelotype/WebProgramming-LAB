โปรแกรมเมอร์ที่ดีจะต้องออกแบบระบบที่เป็นระบบย่อย ๆ สามารถนำกลับมาใช้ใหม่ได้ - ลดความซับซ้อน ลดความผิดพลาด และ ลดงานเมื่อจะต้องมีการแก้ไข


ดูรายละเอียดภาพยนตร์

1. จาก Wireframe ที่ให้มา ให้สร้าง Static HTML Page ตามที่ตัวเองพอใจด้วย HTML + CSS + Bootstrap (ในชั้นตอนนี้ยังไม่จำเป็นต้องมี JS)

2. import Vuejs เข้ามาในหน้าแรก และทดสอบว่า Vuejs ทำงานได้ถูกต้อง

3. Copy ข้อมูลจาก data.js มาไว้ใน data ของ Vue

4. สร้าง Component ชื่อ "movie-card" แสดงรูปภาพโปสเตอร์ของภาพยนตร์เรื่องนั้นๆ (ในส่วนด้านล่างของรูปภาพ Wireframe)

5. ใช้ v-for เพื่อแสดงข้อมูลภาพยนตร์จากข้อมูลใน property: data

6. เมื่อกดที่ component ที่สร้างขึ้นให้แสดง VDO ตัวอย่างหนังจาก youtube.com และ แสดงรายละเอียดของภาพยนตร์ที่เลือก ในส่วนด้านบนดังรูป

7. เมื่อกดปุ่มดูรอบฉายให้ไปที่หน้าจอเลือกที่นั่ง และซื้อตั๋ว -> Homework 3
